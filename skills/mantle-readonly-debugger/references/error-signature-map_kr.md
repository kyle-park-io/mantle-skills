# 오류 서명 맵

이 표를 사용하여 자주 발생하는 읽기 경로 실패를 빠르게 분류하세요.

## RPC 전송/연결

- 증상:
  - 타임아웃
  - 연결 재설정
  - HTTP 429/503
- 가능한 원인:
  - 공급자 포화
  - 속도 제한
  - 불안정한 엔드포인트
- 첫 번째 조치:
  - 백오프로 재시도
  - 폴백 RPC로 전환
  - 배치 크기/요청 비율 줄이기

## 호출/되돌리기 오류

- 증상:
  - `execution reverted`
  - `CALL_EXCEPTION`
  - 데이터가 예상되는 곳에 빈 반환 데이터
- 가능한 원인:
  - 잘못된 컨트랙트 주소
  - 잘못된 ABI/함수 선택자
  - 유효하지 않은 매개변수 또는 상태 사전 조건
- 첫 번째 조치:
  - 대상 주소/ABI 확인
  - 최소 입력으로 드라이런
  - 가능한 경우 되돌리기 데이터 디코딩

## 견적 실패

- 증상:
  - DEX 애그리게이터 견적이 null/오류 반환
  - 경로 없음
  - 출력 금액이 0과 같음
- 가능한 원인:
  - 유동성 부족
  - 지원되지 않는 경로/토큰 쌍
  - 잘못된 소수점/입력 단위
- 첫 번째 조치:
  - 토큰 소수점 및 금액 스케일링 확인
  - 더 작은 크기 시도
  - 대체 경로/소스 시도

## 잔액 불일치

- 증상:
  - 지갑 잔액이 읽기마다 다름
  - 액션 후 예상 잔액이 보이지 않음
- 가능한 원인:
  - 보류 중인 트랜잭션
  - nonce 경합
  - 오래된 블록 태그/소스 지연
- 첫 번째 조치:
  - `latest` vs `pending` 비교
  - 최근 트랜잭션 상태 검사
  - 제한된 지연 후 다시 읽기
